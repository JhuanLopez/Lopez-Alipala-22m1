Central Processing Unit (CPU)

The brain of the computer, responsible for processing instructions.

Motherboard

The main circuit board that connects all the components.

Memory (RAM)

Random Access Memory, used for temporary data storage and quick access.

Storage Drives

  • Hard Disk Drive (HDD): Traditional magnetic storage.
  • Solid State Drive (SSD): Faster, more reliable flash storage.
  • NVMe SSD: A faster type of SSD connected via the PCIe interface.

Power Supply

  • Power Supply Unit (PSU): Converts mains electricity to low-voltage regulated power for the computer components.

Graphics and Display

  • Graphics Processing Unit (GPU): Handles rendering of images, video, and animations.
  • Monitor: Displays the visual output from the computer.

Input Devices

  • Keyboard: For typing and inputting commands.
  • Mouse: For pointing, clicking, and navigating the interface.
  • Touchpad: An alternative to a mouse, often found on laptops.

Additional Input/Output

  • Webcam: For video input.
  • Microphone: For audio input.
  • Speakers: For audio output.
  • Headphones: For private audio output.

Cooling Systems

  • CPU Cooler: Keeps the CPU at an optimal temperature.
  • Air Cooler: Uses fans to cool the CPU.
  • Liquid Cooler: Uses liquid to transfer heat away from the CPU.
  • Case Fans: Help circulate air within the computer case.

Expansion Cards

  • Network Interface Card (NIC): Enables connection to a network.
  • Sound Card: Enhances audio output quality.
  • USB Expansion Card: Provides additional USB ports.

Peripheral Interfaces

  • Ports and Connectors:
  • USB Ports: For connecting peripherals like keyboards, mice, and storage devices.
  • HDMI/DisplayPort: For connecting monitors.
  • Ethernet Port: For wired network connections.
  • Audio Jacks: For connecting audio devices like headphones and microphones.

Computer Case (Chassis)

The enclosure that houses and protects the computer components.

Optical Drives (Less Common Nowadays)

  • DVD/Blu-ray Drive: For reading/writing optical discs.

Miscellaneous

  • BIOS/UEFI Chip: Firmware that initializes hardware during the booting process.
  • Cables and Connectors:
  • Power Cables: Supply power to components.
  • Data Cables: Transfer data between components (e.g., SATA cables).
  • Flash Memory: USB flash drives, SD cards, etc., for additional portable storage.
